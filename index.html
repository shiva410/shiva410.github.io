Animation Reception

This website contains information about the plots and popularity statistics of various Anime. Included are rollover images and a sign-up form. Each input field of the form has validation features and will only be submitted once the user fills out the form properly. Regarding the images, each picture is set up with a rollover feature such that if a user hovers over an image, another image is displayed. HTML, CSS, Javascript, and JQuery were used to build this website.
