Zodiac World

This website was created with HTML, CSS, JavaScript, and jQuery.

Index Page

This contains an overall description of the website, navigation bar, and a sidebar. The navigation bar also has a hover feature.

Pop-Up

When users enter the site, it will show a pop-up asking users to enter their name. After the user enters his/her name, the “Welcome!” message will be modified to include user’s name.

Western Zodiac Symbols

This image is featured on the home page to provide users with an overview of the Western Zodiac Symbols

About Us

In this page, when user clicks the link “Who are we and what is the purpose of this site”, this section expands to show the details of the site.

Eastern Zodiac

This and the Western Zodiac pages include videos with playback controls/sliders to help the user when playing the videos

Contact Us: Pop-Up and Form

Pop-Up – When user clicks “Contact Us” tab, page asks for user’s permission to find his/her location. This page also includes a form for collecting user information, with validation features.

Contact Us: Map

If the user grants permission in the pop-up mentioned above, it will display a google map with the user’s location.
